# 🎧 Online Noise Reducer

A fully online **noise reduction tool** using **Python-only `noisereduce`**.  
Supports both **WAV and MP3**. Works instantly on Hugging Face Spaces.

**How to use:**
1. Click “Upload” to choose your audio (WAV or MP3).
2. Wait a few seconds for processing.
3. Play or download the cleaned audio.

**Tech Stack:** Python + Streamlit + noisereduce

Check out the configuration reference at [Hugging Face Spaces Config](https://huggingface.co/docs/hub/spaces-config-reference)
